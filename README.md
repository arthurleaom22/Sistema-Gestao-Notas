# Sistema-Gestao-Notas
Sistema de Gestão Acadêmica para a disciplina de PDS2 - UFMG. Focado no controle de turmas, alunos e lançamento de notas.

#Descrição do Problema:
A gestão de notas e turmas em ambientes acadêmicos carece de centralização, dificultando o controle de desempenho por parte de professores e diretores. O registro manual pode gerar erros em cálculos de médias e no status final de aprovação.

##Objetivos
Desenvolver um software em C++ para automatizar o gerenciamento escolar, permitindo:
Cadastro de turmas e disciplinas.
Gerenciamento de alunos e professores (POO).
Lançamento de notas com cálculo automático de média e verificação de aprovação.
Tratamento de erros para entradas inválidas.
